# Contributing to Awesome AI Humanizers

First off, thanks for taking the time to contribute! 🎉

## Adding a Tool or Resource

1. **Fork** this repository
2. **Clone** your fork locally
3. **Add** your entry to the appropriate section in `README.md`
   - Keep entries in **alphabetical order** within each section
   - Use the same table format as other entries
   - Provide accurate, concise descriptions
   - Include a direct link to the tool/resource
4. **Commit** with a clear message: `Add [tool name] to [section name]`
5. **Open a Pull Request** with:
   - A brief description of what you're adding
   - Why it belongs on this list
   - Confirm you've verified the tool/resource works

## Guidelines

### For Open-Source Tools
- Must have a working GitHub repo
- Must be actively maintained (commits within last 6 months)
- Include language/framework info
- Star count helps users prioritize

### For Commercial Tools
- Must have a functional website
- Clearly indicate free tier availability
- Accurate pricing descriptions

### For Research Papers
- Must be peer-reviewed or from arXiv
- Include full citation
- Link to the PDF or DOI

## Formatting

Use this template for tools:

```markdown
| [**Tool Name**](https://link) | Language | Brief description. |
```

For research papers:

```markdown
N. **[Paper Title](https://arxiv.org/link)** - Author et al., Year
```

## Code of Conduct

Be respectful, inclusive, and constructive. We're building a helpful resource for everyone.

## Questions?

Open an [issue](https://github.com/rudra496/awesome-ai-humanizers/issues) if you have questions.
